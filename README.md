# 🌸 2026 東京賞櫻之旅 (Tokyo Sakura Trip App)

這是一個專為 **2026 年 3 月東京與箱根賞櫻行** 所打造的「單頁式旅遊手冊 App (Single-file PWA)」。

無需下載、無需登入、無後端伺服器。透過瀏覽器即可運作，並利用 LocalStorage 技術在手機上永久保存你的個人打勾紀錄與記帳資料。

## ✨ 核心功能 (Key Features)

### 📅 智慧行程管理

* **5 天 4 夜完整動線**：包含新宿、箱根（一泊二食）、澀谷、原宿、上野等詳細規劃。
* **🚦 自動導航 (Smart Highlight)**：App 會偵測現在時間，自動將「進行中」的行程標示為粉紅色呼吸燈，打開 App 自動定位到當下行程。
* **備案切換**：Day 2 與 Day 5 內建 A/B 方案（例如：賞櫻 vs 逛街），視天氣與花況一鍵切換。

### 🧰 實用旅遊工具

* **🚕 給司機看 (Taxi Card)**：一鍵跳出超大字體的日文地址與敬語，解決與日本司機的溝通障礙。
* **🗣️ 手指日文發音**：內建 TTS (Text-to-Speech) 引擎，點擊喇叭按鈕即可用標準日語幫你唸出需求。
* **🎫 票券收納夾**：集中管理 Skyliner、飯店訂單號、Visit Japan Web 等資訊，一鍵複製。
* **🗺️ 行李與電梯攻略**：針對攜帶大行李的移動日，提供精準的「電梯入口」與「無障礙路徑」Google Maps 連結。

### 💰 購物與記帳

* **🛍️ 必買清單**：可自定義新增、刪除、勾選戰利品。
* **💸 記帳助手**：快速記錄消費（食/衣/住/行），自動計算總花費並換算台幣參考值。
* **📤 分享功能**：一鍵將購物清單或記帳明細複製為文字，方便貼到 Line 群組備份。

## 📱 如何安裝到手機 (iOS/Android)

為了獲得最佳體驗（全螢幕、無網址列、資料永久儲存），請務必將此網頁加入主畫面。

### iOS (iPhone)

1. 使用 **Safari** 打開網頁。
2. 點擊下方選單中間的 **「分享」** 按鈕 (方框向上箭頭)。
3. 往下滑，選擇 **「加入主畫面 (Add to Home Screen)」**。
4. 點擊右上角的「新增」。
5. 現在桌面上會出現一個櫻花圖示的 App，點開即可使用！

### Android (Chrome)

1. 使用 **Chrome** 打開網頁。
2. 點擊右上角的 **「...」** 選單。
3. 選擇 **「安裝應用程式」** 或 **「加到主畫面」**。

## 🛠️ 技術棧 (Tech Stack)

這是一個極輕量化的專案，所有程式碼都整合在一個 `index.html` 檔案中。

* **Core**: HTML5, Vanilla JavaScript
* **Styling**: [Tailwind CSS](https://tailwindcss.com/) (CDN) - 快速切版與美觀 UI。
* **Logic**: [Alpine.js](https://alpinejs.dev/) (CDN) - 處理資料綁定、Modal 開關與計算邏輯。
* **Interaction**: [Swiper.js](https://swiperjs.com/) (CDN) - 實現如同原生 App 的左右滑動分頁體驗。
* **Icons**: Phosphor Icons.
* **Data**: Browser LocalStorage (資料僅存在使用者手機端，保障隱私)。

## ⚠️ 注意事項

1. **資料隱私**：所有資料（勾選狀態、記帳紀錄）皆儲存於你的手機瀏覽器中，**不會**上傳到任何伺服器。
2. **同步問題**：因為無後端，你與旅伴的手機資料**不會同步**。建議一人負責記帳，或利用「分享」功能互傳文字備份。
3. **iOS Safari 限制**：請務必透過「加入主畫面」的方式使用，若僅在一般網頁模式下瀏覽，Safari 可能會定期清除 LocalStorage 導致紀錄消失。
